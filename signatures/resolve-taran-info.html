How to install this signature in Gmail:
1. Inside the white box below, click anywhere then press Ctrl+A (Windows) or Cmd+A (Mac) to select everything.
2. Copy with Ctrl+C / Cmd+C.
3. Go to Gmail → Settings (gear icon) → See all settings → General → Signature.
4. Create new (call it "Resolve · Info") or edit the existing one.
5. Paste with Ctrl+V / Cmd+V.
6. At the bottom of Settings, under "Signature defaults," set this signature for the address info@resolveproperty.ca on both "FOR NEW EMAILS USE" and "ON REPLY/FORWARD USE".
7. Save Changes at the bottom.
